# Phase 2: PyTorch + ML 코딩

> 기간: ~20주 (약 5개월)
> 목표: PyTorch를 일반 Python처럼 유창하게 쓰고, Transformer를 from scratch 구현

## 디렉토리 구조

```
phase2-pytorch/
├── pytorch-basics/       # PyTorch in One Hour + TensorGym
├── micrograd/            # Karpathy micrograd 재현 (autograd)
├── nanogpt/              # Karpathy nanoGPT 재현 (Transformer)
├── deep-ml/              # Deep-ML 문제풀이
└── numpy-100/            # NumPy 연습
```

## 핵심 마일스톤

1. **micrograd 재현**: chain rule → computational graph → backward pass 이해
2. **nanoGPT 재현**: Embedding → Attention → Block → Decoder 순서로 구현
3. **아무것도 안 보고** MLP, Transformer decoder를 PyTorch로 구현 가능
