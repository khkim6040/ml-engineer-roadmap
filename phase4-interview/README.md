## LeetCode 전략

NeetCode 250 기준, 주 2-3문제. ML 학습하는 날 / LC 하는 날 교대.

**LC 문제를 시스템과 연결하는 관점** (블로그 핵심 통찰):
- Sliding window, two pointers → streaming data processing
- Hash maps, heaps, bucket sort → counting, frequency, heavy hitters
- Tries → inverted indices, autocomplete
- Prefix sums, line sweep → event scanning, interval logic
- Greedy + heap → scheduling, resource allocation
- Union find, graph traversal → data compaction
